Proyek 1: Sistem Monitoring Real-time untuk Kereta Hybrid PNM

Proyek ini, yang merupakan bagian dari Project Matching Fund UNY tahun 2023, adalah kolaborasi dengan PT. Rekaindo Global Jasa. Kami mengembangkan inovasi sistem monitoring real-time untuk meningkatkan pemantauan kondisi kereta hybrid PNM. Sistem ini memanfaatkan Message Queueing Telemetry Transport (MQTT) Broker untuk memberikan akses cepat dan mudah melalui perangkat seluler dan komputer.

Kemampuan untuk memantau berbagai parameter krusial seperti mesin generator, sistem propulsi, dan kecepatan kereta secara real-time. Website ini dapat Meningkatkan standar pemeliharaan dan keamanan kereta, serta kenyamanan penumpang.

Teknologi yang digunakan adalah MQTT Broker dengan broker.hivemq.com, memanfaatkan PLC (Programmable Logic Controller) dan website ini dapatMenampilkan data dalam bentuk angka, gambar, dan grafik sehingga dapat memudahkan dalam monitoring kereta. Data dari berbagai parameter diunggah ke basis data melalui protokol MQTT, kemudian diterima dan ditampilkan oleh website dalam format yang mudah dipahami. Hal ini memungkinkan pengguna untuk memantau kondisi kereta dengan efisien dan praktis.

Proyek 2: HMI Interface Pompa Air Berbasis Haiwell Cloud SCADA

Proyek ini melibatkan pembuatan Human Machine Interface (HMI) untuk sistem pompa air menggunakan software Haiwell Cloud SCADA versi 3.36.9.8. Kami mengintegrasikan PLC Haiwell IOT A8=E dengan protokol MQTT dan menyimpan data dalam basis data MySQL.

Desain HMI yang mudah dipahami dan dioperasikan, memungkinkan pengguna merespons perubahan status dengan cepat. Dilengkapi dengan sistem notifikasi untuk tegangan dan arus yang berada di luar batas normal.

Program ini menghubungkan perangkat dalam jaringan dengan kecepatan dan keandalan tinggi, memungkinkan pertukaran data yang efisien. Datalogger menyimpan data untuk analisis jangka panjang, dan sistem notifikasi memberikan peringatan segera untuk kondisi listrik yang tidak sesuai.

Proyek 3: Sistem Deteksi Kebakaran Berbasis IoT

Proyek ini merancang sistem deteksi kebakaran menggunakan sensor api, sensor suhu DHT11, dan sensor asap MQ-2 dengan Node MCU sebagai mikrokontroler. Sistem ini memanfaatkan aplikasi Blynk untuk memonitor dan mengirimkan notifikasi ke perangkat Android.

Sensor bekerja independen sehingga jika salah satu sensor rusak, sistem tetap berfungsi. Sistem ini Mengirim notifikasi ke perangkat Android dan menampilkan data sensor secara langsung melalui Blynk. Dan juga memberikan peringatan suara ketika terdeteksi kebakaran.

Sistem mengirimkan data sensor ke aplikasi Blynk dan membunyikan alarm ketika terdeteksi kebakaran. Hasil uji coba menunjukkan sistem berjalan dengan baik dan efektif dalam mendeteksi kebakaran.
